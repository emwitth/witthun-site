RemoraFish (My Capstone)

Frontend Github Link

Backend Github Link

RemoraFish is a simple tool for visualizing network activity within a device. It was built using Python3 and Angular is built to run on Linux devices. It provides a view of network activity as a network graph between nodes that represent programs and nodes that represent remote devices. As well, there is more functionality to dive deeper into the activity and hide elements. I will be a copy of my capstone paper here when it is finished (and I graduate).
